## Sobre el Proyecto

En FemCoders Club creemos que la tecnología se construye mejor cuando todas las voces están presentes.
Este proyecto nació como parte de uno de nuestros eventos, con un objetivo claro: crear una experiencia web que combine código, creatividad y comunidad. Cada edición tiene su propia identidad, su propio mensaje y su propia historia — pero todas comparten el mismo espíritu: el de mujeres que se apoyan, aprenden juntas y dejan huella en el mundo tech.
Una interfaz sencilla en su uso, cuidada en su diseño, y con mucho corazón detrás de cada línea de código.

---

## Características

-  Diseño responsive — Se adapta perfectamente a móvil, tablet y escritorio
- Sin dependencias — HTML, CSS y JavaScript puro, sin frameworks ni librerías externas
- Interactividad visual — Animaciones y efectos CSS que invitan a explorar
- Accesible — Construido pensando en todas las personas usuarias
-  Open Source — Abierto a la comunidad para aprender, mejorar y contribuir
- Hecho con intención — Cada evento tiene su historia; este proyecto la cuenta

## Inicio Rápido

### Instalación

1. **Clona el repositorio:**

   ```bash
   git clone https://github.com/femcodersclub/[repo].git
   ```

2. **Accede al directorio:**

   ```bash
   cd [repo]
   ```

3. **Abre en tu navegador:**

   ```bash
   open index.html
   ```

## Contribuir

¿Tienes una idea? ¿Encontraste un bug? ¿Quieres añadir algo? ¡Bienvenida!

1. Haz **fork** del repositorio
2. Crea una rama: `git checkout -b feature/mi-mejora`
3. Haz commit: `git commit -m "feat: descripción del cambio"`
4. Push: `git push origin feature/mi-mejora`
5. Abre una **Pull Request**
